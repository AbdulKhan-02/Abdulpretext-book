Subsubsection 1.1.1

Exploration 1.1.1. Exploring Explorations.

This is an <exploration>. Other similar possibilities are <project>, <activity>, <task>, and <investigation>.
Note that projects, activities, explorations, tasks and investigations share the independent numbering scheme, so it is really only intended you use one of these. If you want a variant of the name (e.g. “Directed Activity”) you can use the <rename> facility ([cross-reference to target(s) "rename-facility" missing or not unique]).
Solution.
This is a “solution” to the exploration. In practice, you might choose to not make this visible for students, but instead include it as part of some guidance you might provide to instructors (e.g. an Instructor’s Manual).
